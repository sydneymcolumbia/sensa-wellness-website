State of Sensa Wellness (static HTML, Vercel, www.sensawellness.org) after the July 2026 GEO overhaul.

**Schema coverage (all JSON-LD validated, 187 blocks site-wide):**

- Every post-*.html (79): Article + BreadcrumbList, all with datePublished. No BlogPosting (Article is used instead, fine).

**Answer-first content:** `.answer-box` component added to styles.css. Placed a 40-60 word extractable direct answer at the top of body on 20 priority posts: understanding-crp, sleep, stress, gut, anti-inflammatory-diet, heart-disease, sugar, exercise, silent, diabetes, inflammatory-foods, aging, obesity, vitamin-d, omega3, intermittent-fasting, longevity, brain, alcohol, smoking.
