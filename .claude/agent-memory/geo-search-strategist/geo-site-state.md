State of Sensa Wellness (static HTML, Vercel, www.sensawellness.org) after the July 2026 GEO overhaul.

**Crawler access:** robots.txt now has explicit `Allow: /` blocks for GPTBot, OAI-SearchBot, ChatGPT-User, PerplexityBot, Perplexity-User, ClaudeBot, Claude-SearchBot, Claude-User, anthropic-ai, Google-Extended, Applebot-Extended, CCBot, cohere-ai. Disallows admin/success/chat/deck/cuvet-animation/start/marketing/node_modules/api. llms.txt exists at root (curated product + page map).

**Schema coverage (all JSON-LD validated, 187 blocks site-wide):**
- Homepage index.html: Organization, WebSite+SearchAction, Product, FAQPage (already existed, not touched — has user's uncommitted manual edits).
- Every post-*.html (79): Article + BreadcrumbList, all with datePublished. No BlogPosting (Article is used instead, fine).
- pay-now.html: Product (3 Offers at 49/99/129 regular price) + BreadcrumbList. NOTE: a live Summer Reset Sale shows 39/79/103 on-page; schema intentionally uses regular prices for durability.
- what-is-crp.html: Article + FAQPage (4 Q&A) + implicit breadcrumb.
- how-to.html: HowTo (6 steps) + BreadcrumbList.
- app.html: SoftwareApplication + BreadcrumbList. our-story.html: AboutPage+Organization + BreadcrumbList. blog.html: Blog + BreadcrumbList. weight-loss.html: Article.

**Answer-first content:** `.answer-box` component (in styles.css). Phase 1 placed extractable direct-answer boxes on 20 priority posts. Phase 2 (July 2026) completed the rest: ALL 79 post-*.html now open with an `.answer-box` (span.answer-label "The short answer" + one 2-4 sentence self-contained `<p>`, inserted right after `<div class="post-body">`). All 5 hub-*.html and all 3 compare-*.html also now have answer-first boxes (hubs: box sits before `.hub-grid`; compares: box in the first `bg-white` intro section before `.two-col`, with inline max-width/margin style).

**FAQPage schema (phase 2):** Added to 5 posts with genuine in-body Q&A only: post-understanding-crp (5 Q&A: what CRP is, normal levels, std vs hsCRP, what raises it, trends), post-silent-inflammation (4), post-inflammation-cause-of-disease (4), post-aging-inflammation (4), post-gut-inflammation (3). Also FAQPage on all 3 compare pages (compare-crp-tests 4 Q&A, compare-inflammation-tests 4, compare-wellness-devices 3). All answers sourced from visible page text. Most other posts use statement-style H2s and did NOT get FAQPage (no genuine multi-question content).

**Hub schema (phase 2):** Each hub-*.html now has CollectionPage schema (name/description/url from page meta + mainEntity ItemList of its cluster post URLs and titles) added after the existing BreadcrumbList block. Verified all hub/compare internal post links resolve on disk (aging 7, chronic-disease 10, diet 11, lifestyle 10, mental-health 7 post links; compares 0-1 each plus the new post-understanding-crp link in compare-crp-tests answer box).

**Sitemap:** sitemap.xml regenerated by gen-sitemap.js (repo root helper) from disk with lastmod/changefreq/priority, 101 URLs. Excludes noindex/utility pages. Re-run `node gen-sitemap.js` after adding pages. (Re-run at end of phase 2, still 101 URLs.)

**Schema type inventory after phase 2:** posts = Article + BreadcrumbList (+ FAQPage on the 5 above). hubs = BreadcrumbList + CollectionPage. compares = BreadcrumbList + Product + FAQPage.

**Audits passed:** 0 broken internal .html links, 0 images missing alt (128 imgs), every indexable page has exactly one H1 (only noindex admin/chat/cuvet-animation differ). Assets sensa-og.jpg, logo.png, favicon.png, theme.css, theme-fx.js all exist.
