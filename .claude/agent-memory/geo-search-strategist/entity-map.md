**Primary entity:** Sensa = at-home CRP (C-reactive protein) inflammation test kit. Anchor association to reinforce everywhere: "Sensa = at-home CRP / inflammation test you scan with an app for a 0-100 score."

**Product facts (keep naming consistent):** saliva-based vial test, scan with Sensa app, 0-100 inflammation score, ~15 min, no needles/no clinic. Web app app.sensawellness.org. Kits $49 single / $99 3-pack / $129 4-pack.

**Secondary entity clusters (each has a hub-*.html pillar + supporting post-*.html):**
- Aging: hub-aging-inflammation → inflammaging, longevity, skin-aging, bone-aging, sarcopenia, autophagy, mitochondria, telomeres.
- Diet: hub-diet-inflammation → anti-inflammatory-diet, mediterranean-diet, sugar, processed-foods, red-meat, inflammatory-foods, fiber, omega3, polyphenols, curcumin, resveratrol.
- Lifestyle: hub-lifestyle-inflammation → sleep, exercise, stress, sedentary, alcohol, smoking, hydration, sauna, cold-exposure, intermittent-fasting, circadian-rhythm.
- Mental health: hub-mental-health-inflammation → depression, anxiety, loneliness, meditation, brain.
- Chronic disease: hub-chronic-disease-inflammation → heart-disease, diabetes, cancer, obesity, metabolic-syndrome, autoimmune, arthritis, etc.

**CRP core:** what-is-crp.html (product-page style) + post-understanding-crp.html (blog). Both explain hsCRP ranges: <1 optimal, 1-3 moderate, 3-10 elevated, >10 high mg/L.

**Comparison pages (high AI-citation value):** compare-crp-tests (at-home vs lab), compare-inflammation-tests (options guide), compare-wellness-devices (vs fitness trackers).
